Sistema avanzado de simulación de producción y saturación de centros de trabajo para la optimización de capacidad y personal (MOD) en entornos industriales.

## 🚀 Características Principales

- **Simulación Dinámica**: Ajuste en tiempo real de OEE, Cadencias (PPM) y Demanda.
- **Datos Reales de Nexus v2**: Conexión directa con los pedidos pendientes para cálculo de saturación actual.
- **Gestión de Escenarios**: Guardado y recuperación de configuraciones de planta.
- **Comparativa de Escenarios**: Análisis visual de variaciones entre modelos de producción.
- **Cálculo de MOD**: Integración del Ratio Persona-Máquina para dimensionamiento de plantilla.
- **Rendimiento Industrial**: Carga de datos vía caché binaria ultra-rápida.

## 🛠️ Guía Rápida de Uso

1. **Explorar**: Usa el **Escenario Base** para ver la saturación actual.
2. **Simular**: Haz clic en **Ajustar** en cualquier artículo para cambiar sus parámetros.
3. **Analizar**: Observa el impacto en el gráfico de saturación y KPIs generales.
4. **Comparar**: Usa el botón **Comparativa** para enfrentar dos realidades distintas.
